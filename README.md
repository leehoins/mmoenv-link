## GitHub Pages setup

1. Push this folder to a GitHub repository.
2. In the repository settings, enable GitHub Pages from the main branch root.
3. Confirm the custom domain is set to `link.mmoenv.com`.
4. In Cloudflare DNS, add the `link` record that GitHub Pages asks for, then wait for HTTPS to finish provisioning.
